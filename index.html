Leetcode 256. Paint House Dynamic Programming - LinkedIn Since no two adjacent houses have the same colors, we need to exclude the same color of previous house. Hence, we compare the costs of color of the current house which has different color than previous Solution: public int minCost(int[][] costs) { // corner cases if(costs==null || costs.length==0 || costs[0].length==0) { return 0; } // we skip the first row (first house) for(int i = 1; i < costs.length; i++) { costs[i][0] += Math.min(costs[i-1][1],costs[i-1][2]); // skip color 0 since it is the same as the previous house costs[i][1] += Math.min(costs[i-1][0],costs[i-1][2]); costs[i][2] += Math.min(costs[i-1][0],costs[i-1][1]); } int n = costs.length-1; return Math.min(Math.min(costs[n][0],costs[n][1]),costs[i][2]); }
